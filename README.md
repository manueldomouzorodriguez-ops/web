# Web personal

Sitio personal y portfolio en HTML + CSS.

## Páginas

- `index.html`: home personal (sobre mí, servicios y contacto).
- `portfolio.html`: selección de proyectos.

## Uso

Abre `index.html` directamente en el navegador o sirve el contenido con:

```bash
python3 -m http.server 8000
```
